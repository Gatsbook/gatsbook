# Gatsbook

## Getting Started

You'll need Node.js and Gatsby CLI

```bash
gatsby new project-name https://github.com/gatsbook/gatsbook
cd project-name
npm run develop
```

## Building your site

```bash
npm run build
```

## Configuration

- Site configurations is located `config/website.js`
